# 如何啟用梅林路由器的 SSH

本教程将会说明如何启用梅林路由器的 SSH 功能。



## 操作流程

1. 打开并登入梅林路由器的后台
2. 定位到【**系统管理**】》【**系统设置**】
3. 寻找【**SSH Daemon**】
4. 修改 Enable SSH 设定到 **LAN Only**
5. 修改 Allow SSH password login 到“是”
6. 到页面底部点击【**应用本页面设置**】
