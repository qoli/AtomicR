# Plane Router

![](https://img.shields.io/discord/551675706749353995.svg)



Plane Router 是一款基於 JSON 腳本運作的 App。

可用於管理路由器、樹莓派、服務器等透過 SSH 訪問的設備。



AtomicR 項目是 PlaneR 的腳本存放位置。
